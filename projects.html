프로젝트 1: 부모 정신 건강 요인이 아동 행복에 미치는 영향 분석 및 정책 제안

회귀 모델 기반 아동 행복 지수 주요 변수 탐색 (Python, Linear/Lasso/Random Forest)

1. 배경 및 분석 목표

최근 부모의 정신적 안정과 부부간의 상호 지지도가 아동의 행복 및 정서 발달에 미치는 영향에 대한 관심이 증가하고 있습니다. 본 프로젝트는 데이터 기반으로 아동 행복 수준에 영향을 미치는 부모 및 가족 요인을 정량적으로 파악하고, 이를 바탕으로 기존 정책 방향에 대한 개선안을 제안하는 것을 목표로 합니다.

  • 가설 1: 부부간 상호 지지도가 높을수록 아동의 행복에 큰 영향을 미칠 것이다.
  • 가설 2: 가족 전체의 지지도가 높을수록 아동의 행복 수준이 올라갈 것이다.

2. 분석 데이터 및 주요 변수

한국청소년정책연구원의 '한국아동패널 2022년 설문지 자료(어머니, 아버지, 아동)'를 활용했습니다. (표본 규모: 약 2,150가구, 총 1,469개 변수)

3. 데이터 분석 과정

  • 데이터 전처리: 결측치 제거, 변수 표준화
  • 탐색적 분석 (EDA): 주요 변수 간 상관분석 및 시각화
  • 모델링: Linear, Lasso, Random Forest 회귀 모델 비교
  • 결과 해석: 피처 중요도 및 R² 값 분석

4. 분석 결과 및 정책 제안

  • 기존 정책

    제2차 아동정책기본계획(2020~2025년)을 기반으로, 주로 아동의 권리 보장과 놀이·학습 중심의 정책에 초점을 맞춤.

  • 보완점

    1. 가족 간의 의사소통과 정서적 교류 강화 필요

    2. 아동이 자신의 생각을 표현할 수 있는 독립적인 소통 창구 마련

  • 정책 제안

    1. 가족 소통 행복 지수 의무 측정 제도화

    2. 학교 단위의 아동 행복 지수 정기적 측정

    3. 가족센터 및 지역사회 기관과의 연계 강화

  • 기대 효과

    1. 부모가 아동의 정서적 상태를 객관적으로 파악 가능

    2. 가정 내 의사소통 부족 인식 및 개선 유도

    3. 가족 간 상호이해와 정서적 유대 강화

    4. 아동의 정서적 안정과 행복 수준 향상

5. 보안점

아동 행복에 미치는 요인을 더 자세하게 분석하기 위해선 아동의 또래 관계, 부모의 지위/사회적 요인 등 다양한 요인들을 종합적으로 분석해야함.

프로젝트 2: 고객 맞춤형 카드 추천 챗봇 서비스

사용자 소비 패턴 기반 카드 추천 (Python, LLM, LangChain)

1. 배경 및 가설

고객의 생활 패턴과 선호 스타일을 반영하여 카드 추천을 자동화하는 챗봇을 구현했습니다.

2. 분석 데이터

카드고릴라 웹 데이터를 크롤링하여 카드 이미지, 혜택, 신청 링크 등을 정제한 JSON 파일을 구축했습니다.

3. 데이터 분석 과정

  • 데이터 전처리 및 혜택 분류
  • LangChain 기반 RAG 구조 구성
  • 사용자 페르소나 입력 → 카드 추천

4. 분석 결과 및 서비스 제안

추천된 카드는 사용자의 소비 패턴과 선호 스타일에 맞춰 이미지·혜택·신청 링크가 자동 제공됩니다. 향후 실사용자 피드백 기반으로 알고리즘 고도화를 추진합니다.

프로젝트 3: 소셜 미디어를 활용한 공공 보건 캠페인의 확산 효과 측정

캠페인 관련 여론 분석 및 미디어 효과 검증 (Python, NLP, Tableau)

1. 배경 및 가설

공공 보건 캠페인의 실효성을 데이터로 검증하기 위해 소셜 미디어 반응을 분석했습니다.

2. 분석 데이터

캠페인 기간 전후 주요 플랫폼(X, 블로그, 커뮤니티)의 텍스트 데이터 약 5만 건을 수집했습니다.

3. 데이터 분석 과정

  • 형태소 분석 및 감성 분류
  • 연관어 네트워크 분석 및 확산 경로 시각화
  • Tableau 대시보드를 통한 일별 언급량 및 감성 추이 비교

4. 분석 결과 및 정책 제안

TV 광고보다 커뮤니티·유튜브형 콘텐츠가 감정적 공감을 더 유발함을 확인하여 예산 재배분을 제안했습니다.

프로젝트 4: 대중교통 이용 패턴 기반 도시 환경 만족도 예측 모델 개발

대중교통 이용 데이터와 환경 데이터를 결합한 만족도 예측 (Python, 시계열 분석)

1. 배경 및 가설

도시민의 생활 패턴에서 환경 만족도를 추론하고자 대중교통과 미세먼지 데이터를 결합했습니다.

2. 분석 데이터

교통카드 데이터, 미세먼지 지수, 공원 접근성(GIS)을 결합한 시계열 데이터를 사용했습니다.

3. 데이터 분석 과정

  • 시계열 통합 및 정규화
  • 출퇴근 패턴과 환경 요인 간 상관관계 분석
  • ARIMA 기반 만족도 예측 모델 구축

4. 분석 결과 및 정책 제안

미세먼지가 나쁨 이상일 때 버스 이용률이 급감하여, 지하철 중심보다 버스 노선 기반 대기질 개선이 필요함을 제안했습니다.
